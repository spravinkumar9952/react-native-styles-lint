# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2025-12-17

### Added

- Added repository, bugs, and homepage URLs to package.json
- Added example project with setup scripts for ESLint 8.x compatibility
- Added setup script for automatic plugin wrapper creation

### Changed

- Updated package metadata with GitHub repository information
- Improved documentation with clearer installation and usage instructions

## [0.1.0] - 2025-12-17

### Added

- Initial release of `@spravinkumar9952/react-native-styles-lint`
- ESLint rule `no-restricted-styles` for detecting restricted style properties in React Native `StyleSheet.create()`
- Support for auto-fixing restricted style properties
- Support for nested style objects
- Case-insensitive matching for style property names
- Support for both camelCase and kebab-case property names
